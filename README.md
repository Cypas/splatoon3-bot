## 自己搭建指南

python >= 3.9
1. python -m pip install --user pipx
2. python -m pipx ensurepath
3. pipx install nb-cli
4. python -m venv .venv
5. source .venv/bin/activate
6. pip install -r requirements.txt
7. cp .env.sample .env
8. edit .env
9. nb run --reload
